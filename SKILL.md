# Report Creator - 技术调研报告撰写

本Skill帮助用户完成从技术主题搜索到演示文档生成的完整技术调研报告撰写流程。

## 工作流程

### 阶段1：信息搜集

1. **分析用户需求**
   - 确定技术主题/领域
   - 识别相关关键词（中英文）
   - 确定搜索范围和时间范围

2. **多渠道搜索**
   使用SearchWeb工具搜索以下信息源：
   - 技术博客（Medium、Dev.to、知乎技术专栏等）
   - 科技网站（TechCrunch、The Verge、36氪等）
   - 学术论文（arXiv、Google Scholar等）
   - 领域关键人物社交媒体（Twitter/X、LinkedIn等）
   - 开源社区（GitHub、Hacker News等）

3. **搜索关键词策略**
   - 核心技术词：如 "programming agents", "coding agents", "AI programming assistants"
   - 趋势词：如 "2024 trends", "future of", "state of the art"
   - 竞品对比词：如 "vs", "comparison", "benchmark"
   - 中文关键词：如 "编程智能体", "代码生成AI", "AI编程助手"

### 阶段2：信息分析与整理

将搜集到的信息按以下维度整理：

1. **技术发展历史**
   - 技术起源与早期发展
   - 关键里程碑事件
   - 发展阶段划分

2. **当前竞品对比**
   - 主要厂商/产品列表
   - 功能特性对比矩阵
   - 性能基准测试结果
   - 市场份额/用户规模

3. **关键技术解析**
   - 核心技术原理
   - 技术架构图
   - 关键算法/模型
   - 技术难点与挑战

4. **未来发展趋势**
   - 技术演进方向
   - 潜在应用场景
   - 行业预测数据
   - 风险与挑战

### 阶段3：撰写Markdown报告

1. **报告结构**
   ```markdown
   # 《{技术主题}技术调研报告》

   ## 摘要
   - 研究背景与目的
   - 主要发现概述
   - 核心结论

   ## 1. 引言
   - 研究背景
   - 研究范围与方法
   - 报告结构

   ## 2. 技术发展历史
   - 2.1 起源与早期发展
   - 2.2 关键里程碑
   - 2.3 发展阶段

   ## 3. 市场现状与竞品分析
   - 3.1 主要参与者
   - 3.2 产品对比
   - 3.3 市场份额

   ## 4. 关键技术解析
   - 4.1 技术原理
   - 4.2 架构分析
   - 4.3 核心挑战

   ## 5. 未来发展趋势
   - 5.1 技术演进方向
   - 5.2 应用前景
   - 5.3 风险与挑战

   ## 6. 总结与建议
   - 核心发现
   - 战略建议
   - 参考资料
   ```

2. **内容质量要求**
   - 数据准确，注明来源
   - 观点客观，多方印证
   - 图表清晰，标注来源
   - 逻辑清晰，层次分明

### 阶段4：生成LaTeX Beamer演示文档

1. **使用模板**
   - 模板位置：`assets/beamer-template.tex`
   - 使用ctex宏包支持中文
   - 采用适合技术报告的主题风格

2. **转换流程**
   - 从Markdown提取关键内容
   - 填充到Beamer模板
   - 生成`.tex`文件

3. **幻灯片结构**
   ```
   - 标题页
   - 目录
   - 研究背景
   - 技术发展历程（时间线形式）
   - 市场现状（图表+要点）
   - 竞品对比（表格形式）
   - 关键技术（架构图+要点）
   - 未来趋势
   - 总结与展望
   - Q&A/致谢
   ```

### 阶段5：编译PDF

1. **编译命令**
   ```bash
   xelatex -interaction=nonstopmode {filename}.tex
   xelatex -interaction=nonstopmode {filename}.tex  # 编译两次以正确生成目录
   ```

2. **常见问题处理**
   - 中文字体问题：确保使用xelatex编译
   - 图片路径：使用相对路径
   - 表格过宽：使用`\resizebox`缩放

## 执行示例

**用户输入**："帮我就编程智能体技术领域撰写技术调研报告"

**执行步骤**：
1. 搜索关键词："coding agents", "AI programming assistants", "programming agents 2024", "GitHub Copilot vs"
2. 阅读分析搜索结果，提取关键信息
3. 按报告结构撰写Markdown文档
4. 使用beamer模板生成演示文档
5. 编译生成PDF

## 输出要求

1. **Markdown报告**
   - 文件名：`{主题}_技术调研报告.md`
   - 内容完整，结构清晰
   - 包含数据图表和引用来源

2. **演示文档**
   - 文件名：`{主题}_技术调研报告.tex`
   - PDF文件名：`{主题}_技术调研报告.pdf`
   - 15-25页为宜
   - 图文并茂，重点突出
